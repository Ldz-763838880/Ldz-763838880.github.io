1091-二进制矩阵中的最短路径

2023-05-26
  • BFS
  • leetcode

1 题目描述

给你一个 n x n 的二进制矩阵 grid 中，返回矩阵中最短 畅通路径 的长度。如果不存在这样的路径，返回 -1 。

二进制矩阵中的 畅通路径 是一条从 左上角 单元格（即，(0, 0)）到 右下角 单元格（即，(n - 1, n - 1)）的路径，该路径同时满足下述要求：

路径途经的所有单元格都的值都是 0 。
路径中所有相邻的单元格应当在 8 个方向之一 上连通（即，相邻两单元之间彼此不同且共享一条边或者一个角）。
畅通路径的长度 是该路径途经的单元格总数。

示例

输入：grid = [[0,1],[1,0]]
输出：2

链接

题目来源：力扣（LeetCode）1091.二进制矩阵中的最短路径

2 解题思路

  • 经典BFS
    • 首先判断出发点是否为0，不是则不存在路径返回-1
    • 然后用队列进行BFS遍历所有节点，同时初始化一个二维数组dist记录各个节点的最短路径，初始化为无穷大，dist[0][0] = 1
    • 接着以队列是否为空为条件遍历节点，每次弹出一个节点，判断周围节点下标是否符合条件，以及节点值是否为0，并用当前节点值是否等于前一个节点 + 1 来确定是否被访问过
    • 遍历时当队列出栈的节点为右下角节点时，返回dist[n - 1][n - 1]即可，否则继续遍历并更新节点路径值，同时将下一个满足路径的节点入队列
    • 循环结束则返回 - 1

3 复杂度分析

  • 时间复杂度
    • O(n ^ 2)
    • 最多将n X n大小的矩阵元素都访问一遍
  • 空间复杂度
    • O(n ^ 2)
    • 需要一个n X n的数组记录每个节点的最短路径值

4 代码

// BFS
class Solution:
    def shortestPathBinaryMatrix(self, grid: List[List[int]]) -> int:
        if grid[0][0] == 1:
            return -1
        n = len(grid)
        dist = [[inf] * n for _ in range(n)]
        dist[0][0] = 1
        queue = deque([(0, 0)])
        while queue:
            x, y = queue.popleft()
            if x == y == n - 1:
                return dist[x][y]
            for i in range(-1, 2):
                for j in range(-1, 2):
                    if x + i < 0 or x + i >= n or y + j < 0 or y + j >= n:
                        continue
                    if grid[x + i][y + j] == 1 or dist[x + i][y + j] <= dist[x][y] + 1:
                        continue
                    dist[x + i][y + j] =  dist[x][y] + 1
                    queue.append((x + i, y + j))
        return -1

1054-距离相等的条形码

1 题目描述

在一个仓库里，有一排条形码，其中第 i 个条形码为 barcodes[i]。

请你重新排列这些条形码，使其中任意两个相邻的条形码不能相等。 你可以返回任何满足该要求的答案，此题保证存在答案。

示例

输入：barcodes = [1,1,1,2,2,2]
输出：[2,1,2,1,2,1]

链接

题目来源：力扣（LeetCode）1054.距离相等的条形码

2 解题思路

  • 最大堆
    • 首先考虑从数量最多的元素开始添加，用最大堆存储元素值和数量
    • 然后每次推出最大数量的元素，如果和上个添加元素不同则直接添加并更新最大堆，相同则推出第二数量的元素添加并更新堆
    • 最后输出res即可

3 复杂度分析

  • 时间复杂度
    • O(n log（n）)
    • n个元素需要插入，同时每次插入调整堆的时间复杂度为log（n）
  • 空间复杂度
    • O(n)
    • 一共有n个元素

4 代码

// 最大堆
class Solution:
    def rearrangeBarcodes(self, barcodes: List[int]) -> List[int]:
        cnt = Counter(barcodes)
        q = []
        for num,c in cnt.items():
            heapq.heappush(q, (-c, num))
        res = []
        while len(q) > 0:
            cx, num = heapq.heappop(q)
            if len(res) == 0 or res[-1] != num:
                res.append(num)
                if cx < -1:
                    heapq.heappush(q, (cx + 1, num))
            else:
                cy, numy = heapq.heappop(q)
                res.append(numy)
                if cy < -1:
                   heapq.heappush(q, (cy + 1, numy))
                heapq.heappush(q, (cx, num))
        return res

1 题目描述

给出一个单词数组 words ，其中每个单词都由小写英文字母组成。

如果我们可以 不改变其他字符的顺序 ，在 wordA 的任何地方添加 恰好一个 字母使其变成 wordB ，那么我们认为 wordA 是 wordB 的 前身 。

例如，”abc” 是 “abac” 的 前身 ，而 “cba” 不是 “bcad” 的 前身
词链是单词 [word_1, word_2, …, word_k] 组成的序列，k >= 1，其中 word1 是 word2 的前身，word2 是 word3 的前身，依此类推。一个单词通常是 k == 1 的 单词链 。

从给定单词列表 words 中选择单词组成词链，返回 词链的 最长可能长度 。

示例

输入：words = ["a","b","ba","bca","bda","bdca"]
输出：4
解释：最长单词链之一为 ["a","ba","bda","bdca"]

链接

题目来源：力扣（LeetCode）1048.最长字符串链

2 解题思路

  • 动态规划
    • 首先考虑以单词word为结尾的词链，则前一个词必然是word减一个单词的，每个单词的初始最大词链长度都为1
    • 然后考虑使用字典cnt存储单词word为尾部词链结尾时的最大长度，通过减少当前word的字母去查找前一个单词是否cnt出现过，出现过则最大长度须更新
    • 动态规划递归方程为：cnt[word] = max(cnt[word], cnt[pre_word]+1],pre_word = {word依次减少一个字母集合}
    • 最后使用中间变量比较每个cnt[word]并记录最大值即可

3 复杂度分析

  • 时间复杂度
    • O(n X m X (log n + m))
    • n代表单词个数，m表示单词平均长度，排序时间复杂度n X m X logn，每个单词需要查询其前身并比较时间复杂度n X m X m
  • 空间复杂度
    • O(n X m)
    • 存储n个单词，每个单词平均长度m，需要空间n X m

4 代码

// DP
class Solution:
    def longestStrChain(self, words: List[str]) -> int:
        cnt = defaultdict(int)
        words.sort(key = len)
        res = 0
        for word in words:
            cnt[word] = 1
            for i in range(len(word)):
                pre = word[:i] + word[i + 1:]
                if pre in cnt:
                    cnt[word] = max(cnt[word], cnt[pre] + 1)
            res = max(res, cnt[word])
        return res

1105-填充书架

2023-04-24
  • leetcode

1 题目描述

给定一个数组 books ，其中 books[i] = [thicknessi, heighti] 表示第 i 本书的厚度和高度。你也会得到一个整数 shelfWidth 。

按顺序 将这些书摆放到总宽度为 shelfWidth 的书架上。

先选几本书放在书架上（它们的厚度之和小于等于书架的宽度 shelfWidth ），然后再建一层书架。重复这个过程，直到把所有的书都放在书架上。

需要注意的是，在上述过程的每个步骤中，摆放书的顺序与你整理好的顺序相同。

例如，如果这里有 5 本书，那么可能的一种摆放情况是：第一和第二本书放在第一层书架上，第三本书放在第二层书架上，第四和第五本书放在最后一层书架上。
每一层所摆放的书的最大高度就是这一层书架的层高，书架整体的高度为各层高之和。

以这种方式布置书架，返回书架整体可能的最小高度。

示例

输入：books = [[1,1],[2,3],[2,3],[1,1],[1,1],[1,1],[1,2]], shelfWidth = 4
输出：6
解释：
3 层书架的高度和为 1 + 3 + 2 = 6 。
第 2 本书不必放在第一层书架上。

链接

题目来源：力扣（LeetCode）1105.填充书架

2 解题思路

  • 动态规划
    • 首先考虑使用dp存储放到第i本书时的最低高度
    • 其次考虑按照书的顺序进行遍历，当放到第i本书时，前j本书的最低高度dp[j] + j <= k <= i 区间书的最大高度就是当前第i本书的最低高度值
    • 动态规划递归方程为：dp[i] = min(dp[i], dp[j] + maxHig(j<= k <= i)]
    • 最后返回dp[n]就是n本书的最低高度值

3 复杂度分析

  • 时间复杂度
    • O(n^2）
    • n本书，每本书需要遍历前面的书
  • 空间复杂度
    • O(n)
    • 存储n本书的最低高度数组dp

4 代码

// DP
class Solution:
    def minHeightShelves(self, books: List[List[int]], shelfWidth: int) -> int:
        n = len(books)
        dp = [inf] *(n + 1)
        dp[0] = 0
        for i, b in enumerate(books):
            curWid, maxHig = 0, 0
            j = i
            while j >= 0:
                curWid += books[j][0]
                if curWid > shelfWidth:
                    break
                maxHig = max(maxHig, books[j][1])
                dp[i + 1] = min(dp[i + 1], dp[j] + maxHig)
                j -= 1
        return dp[n]

1027-最长等差数列

2023-04-22
  • leetcode
  • 动态规划

1 题目描述

给你一个整数数组 nums，返回 nums 中最长等差子序列的长度。

回想一下，nums 的子序列是一个列表 nums[i1], nums[i2], …, nums[ik] ，且 0 <= i1 < i2 < … < ik <= nums.length - 1。并且如果 seq[i+1] - seq[i]( 0 <= i < seq.length - 1) 的值都相同，那么序列 seq 是等差的。

示例

输入：nums = [3,6,9,12]
输出：4
解释：
整个数组是公差为 3 的等差数列。

链接

题目来源：力扣（LeetCode）1027.最长等差数列

2 解题思路

  • 动态规划
    • 首先可以确定等差子序列的公差范围是[-diff, diff] , diff = max(nums) - min(nums)，可以使用字典f来存储以num为结尾的最长子序列长度
    • 其次考虑访问到数组第i个元素时如果将当前num作为最后一个等差元素加入子序列，则f[num] = max(f[num], f[num - d] + 1)，若不加入则f[num] = max(f[num], 1)
    • 最后在遍历中用ans记录f[num]的最大值即可

3 复杂度分析

  • 时间复杂度
    • O(nC）
    • 数组有n个元素，等差子序列公差范围为C
  • 空间复杂度
    • O(n)
    • 哈希表f的空间为n个元素

4 代码

// DP
class Solution:
    def longestArithSeqLength(self, nums: List[int]) -> int:
        minv, maxv = min(nums), max(nums)
        diff = maxv - minv
        ans = 1

        for d in range(-diff, diff + 1):
            f = dict()
            for num in nums:
                if (prev := num - d)  in f:
                    f[num] = max(f.get(num, 0), f[prev] + 1)
                    ans = max(ans, f[num])
                f[num] = max(f.get(num, 0), 1)
        return ans

1043-分隔数组以得到最大和

2023-04-19
  • leetcode
  • 动态规划

1 题目描述

给你一个整数数组 arr，请你将该数组分隔为长度 最多 为 k 的一些（连续）子数组。分隔完成后，每个子数组的中的所有值都会变为该子数组中的最大值。

返回将数组分隔变换后能够得到的元素最大和。本题所用到的测试用例会确保答案是一个 32 位整数。

示例

输入：arr = [1,15,7,9,2,5,10], k = 3
输出：84
解释：数组变为 [15,15,15,9,10,10,10]

链接

题目来源：力扣（LeetCode）1043.分隔数组以得到最大和

2 解题思路

  • 动态规划
    • 考虑将第i个元素当作当前数组最后一个元素进行划分，从第[i - k, i - 1]范围内枚举arr[j],用maVal记录以arr[j]结尾的最大数组值并求和从而进行更新d[i]
    • 状态转移方程：d[i] = max(d[i], d[j] + maxVal * (i - j))

3 复杂度分析

  • 时间复杂度
    • O(nk）
    • 数组有n个元素，每个元素倒序遍历前面k个元素
  • 空间复杂度
    • O(n)
    • dp数组空间大小n，记录对应位置的最大数组求和值

4 代码

// DP
class Solution:
    def maxSumAfterPartitioning(self, arr: List[int], k: int) -> int:
        n = len(arr)
        d = [0] * (n + 1)
        for i in range(1, n + 1):
            maxVal = arr[i - 1]
            for j in range(i - 1, max(-1, i - k - 1), -1):
                d[i] = max(d[i], d[j] + maxVal * (i - j))
                if j > 0:
                    maxVal = max(maxVal, arr[j - 1])
        return d[n]

1026-节点与其祖先之间的最大差值

2023-04-18

  • leetcode

1 题目描述

给定二叉树的根节点 root，找出存在于 不同 节点 A 和 B 之间的最大值 V，其中 V = |A.val - B.val|，且 A 是 B 的祖先。

（如果 A 的任何子节点之一为 B，或者 A 的任何子节点是 B 的祖先，那么我们认为 A 是 B 的祖先）

示例

输入：root = [8,3,10,1,6,null,14,null,null,4,7,13]
输出：7
解释： 
我们有大量的节点与其祖先的差值，其中一些如下：
|8 - 3| = 5
|3 - 7| = 4
|8 - 1| = 7
|10 - 13| = 3
在所有可能的差值中，最大值 7 由 |8 - 1| = 7 得出。

链接

题目来源：力扣（LeetCode）1026.节点与其祖先之间的最大差值

2 解题思路

  • DFS
    • 直接用DFS搜索二叉树的路径的所有节点，同时记录当前路径下成立父子关系节点的最大值ma和最小值mi
    • 统计每个节点对当前路径下mi与ma的绝对差值大小取最大值，最后DFS返回该最大值即可

3 复杂度分析

  • 时间复杂度
    • O(n）
    • 二叉树有n个节点需要遍历
  • 空间复杂度
    • O(n)
    • 最坏情况下，二叉树退化为链表，递归栈的空间为 O(n)

4 代码

// DFS
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxAncestorDiff(self, root: Optional[TreeNode]) -> int:
        def dfs(root, mi, ma):
            if root == None:
                return 0
            diff = max(abs(root.val - mi), abs(root.val - ma))
            mi = min(mi, root.val)
            ma = max(ma, root.val)
            diff = max(diff, dfs(root.left, mi, ma))
            diff = max(diff, dfs(root.right, mi, ma))
            return diff
        return dfs(root, root.val, root.val)

Alice 和 Bob 计划分别去罗马开会。

给你四个字符串 arriveAlice ，leaveAlice ，arriveBob 和 leaveBob 。Alice 会在日期 arriveAlice 到 leaveAlice 之间在城市里（日期为闭区间），而 Bob 在日期 arriveBob 到 leaveBob 之间在城市里（日期为闭区间）。每个字符串都包含 5 个字符，格式为 “MM-DD” ，对应着一个日期的月和日。

请你返回 Alice和 Bob 同时在罗马的天数。

你可以假设所有日期都在 同一个 自然年，而且 不是 闰年。每个月份的天数分别为：[31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31] 。

示例

输入：arriveAlice = "08-15", leaveAlice = "08-18", arriveBob = "08-16", leaveBob = "08-19"
输出：3
解释：Alice 从 8 月 15 号到 8 月 18 号在罗马。Bob 从 8 月 16 号到 8 月 19 号在罗马，他们同时在罗马的日期为 8 月 16、17 和 18 号。所以答案为 3 。

链接

题目来源：力扣（LeetCode）2409.统计共同度过的日子数

2 解题思路

  • 前缀和
    • 利用前缀和统计出每个月对应该年的天数，再加上天数得到日期具体是该年的哪一天
    • 用二个人离开日子的最小值减去到达日子的最大值即得到共同呆在罗马的日子

3 复杂度分析

  • 时间复杂度
    • O(1）
    • 前缀和处理12个月对应天数，时间复杂度为常数
  • 空间复杂度
    • O(1)
    • 一年只有12个月，存储这些天数空间复杂度为常数

4 代码

// 前缀和模拟
class Solution:
    def countDaysTogether(self, arriveAlice: str, leaveAlice: str, arriveBob: str, leaveBob: str) -> int:
        datesOfMonths = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
        prefixSum = [0]
        for date in datesOfMonths:
            prefixSum.append(prefixSum[-1] + date)

        arriveAliceDay = self.calculateDayOfYear(arriveAlice, prefixSum)
        leaveAliceDay = self.calculateDayOfYear(leaveAlice, prefixSum)
        arriveBobDay = self.calculateDayOfYear(arriveBob, prefixSum)
        leaveBobDay = self.calculateDayOfYear(leaveBob, prefixSum)

        return max(0, min(leaveAliceDay, leaveBobDay) - max(arriveAliceDay, arriveBobDay) + 1)


    def calculateDayOfYear(self, day: str, prefixSum: List[int]) -> int:
        month = int(day[:2])
        day = int(day[3:])
        return prefixSum[month - 1] + day

1157-子数组中占绝大多数的元素

2023-04-16
  • leetcode
  • 二分查找
  • 线段树
  • 随机化

1 题目描述

设计一个数据结构，有效地找到给定子数组的 多数元素 。

子数组的 多数元素 是在子数组中出现 threshold 次数或次数以上的元素。

实现 MajorityChecker 类:

MajorityChecker(int[] arr) 会用给定的数组 arr 对 MajorityChecker 初始化。
int query(int left, int right, int threshold) 返回子数组中的元素  arr[left…right] 至少出现 threshold 次数，如果不存在这样的元素则返回 -1。

链接

题目来源，力扣（LeetCode）1157.子数组中占绝大多数的元素

2 解题思路

  • 随机化
    • 题目本质是求区间[left, right]之间的众数
    • 众数出现次数>=区间长度，所以如果随机抽样该区间内的数，每次抽中众数的概率>= 1/2
    • 假设随机抽取30次，则一次都没抽中众数的概率<= (1/2)^k
    • 如果进行p次查询，每次查询相互独立，每次抽取k次则这p次查询结果都正确的概率为$$(1 - (\frac{1}{2})^k)^p.$$，当p = 10^4， k = 30时计算该表达式的结果近似为99.99%
    • 故认为该随机抽样得到区间[left, right]的正确性可以保证
  • 二分查找
    • 根据随机化原理得到区间的众数值x后，需要统计其出现的具体次数
    • 考虑使用哈希字典存储arr，统计每个元素出现的位置序列list
    • 在list中使用二分查找边界right和left出现的下标，相减后得到其出现次数cnt
    • 当cnt >= threshold时满足条件，如果cnt >= 区间长度的一半但却小于threshold则返回-1

3 复杂度分析

  • 时间复杂度
    • O(n + kqlog n）
    • 哈希表处理时间复杂度O(n),共进行q轮查询，每轮k次，每次二分的时间复杂度为O(log n)
  • 空间复杂度
    • 哈希存储空间复杂度O(n)

4 代码

// 随机化 + 二分查找
class MajorityChecker:
    k = 30

    def __init__(self, arr: List[int]):
        self.arr = arr
        self.cnt = defaultdict(list)
        for i, a in enumerate(arr):
            self.cnt[a].append(i)
            

    def query(self, left: int, right: int, threshold: int) -> int:
        arr_ = self.arr
        cnt_ = self.cnt

        length = right - left + 1
        for i in range(MajorityChecker.k):
            x = arr_[randint(left, right)]
            pos = cnt_[x]
            occ = bisect_right(pos, right) - bisect_left(pos, left)
            if occ >= threshold:
                return x
            elif occ * 2 >= length:
                return -1
        return -1

// 官方解法2：摩尔投票 + 线段树
class Node:
    def __init__(self, x: int = 0, cnt: int = 0):
        self.x = x
        self.cnt = cnt
    
    def __iadd__(self, that: "Node") -> "Node":
        if self.x == that.x:
            self.cnt += that.cnt
        elif self.cnt >= that.cnt:
            self.cnt -= that.cnt
        else:
            self.x = that.x
            self.cnt = that.cnt - self.cnt
        return self

class MajorityChecker:
    def __init__(self, arr: List[int]):
        self.n = len(arr)
        self.arr = arr
        self.loc = defaultdict(list)

        for i, val in enumerate(arr):
            self.loc[val].append(i)
        
        self.tree = [Node() for _ in range(self.n * 4)]
        self.seg_build(0, 0, self.n - 1)

    def query(self, left: int, right: int, threshold: int) -> int:
        loc_ = self.loc

        ans = Node()
        self.seg_query(0, 0, self.n - 1, left, right, ans)
        pos = loc_[ans.x]
        occ = bisect_right(pos, right) - bisect_left(pos, left)
        return ans.x if occ >= threshold else -1
    
    def seg_build(self, idx: int, l: int, r: int):
        arr_ = self.arr
        tree_ = self.tree

        if l == r:
            tree_[idx] = Node(arr_[l], 1)
            return
        
        mid = (l + r) // 2
        self.seg_build(idx * 2 + 1, l, mid)
        self.seg_build(idx * 2 + 2, mid + 1, r)
        tree_[idx] += tree_[idx * 2 + 1]
        tree_[idx] += tree_[idx * 2 + 2]

    def seg_query(self, idx: int, l: int, r: int, ql: int, qr: int, ans: Node):
        tree_ = self.tree

        if l > qr or r < ql:
            return
        
        if ql <= l and r <= qr:
            ans += tree_[idx]
            return

        mid = (l + r) // 2
        self.seg_query(idx * 2 + 1, l, mid, ql, qr, ans)
        self.seg_query(idx * 2 + 2, mid + 1, r, ql, qr, ans)
