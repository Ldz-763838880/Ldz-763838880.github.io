1043-分隔数组以得到最大和

2023-04-19
  • leetcode
  • 动态规划

1 题目描述

给你一个整数数组 arr，请你将该数组分隔为长度 最多 为 k 的一些（连续）子数组。分隔完成后，每个子数组的中的所有值都会变为该子数组中的最大值。

返回将数组分隔变换后能够得到的元素最大和。本题所用到的测试用例会确保答案是一个 32 位整数。

示例

输入：arr = [1,15,7,9,2,5,10], k = 3
输出：84
解释：数组变为 [15,15,15,9,10,10,10]

链接

题目来源：力扣（LeetCode）1043.分隔数组以得到最大和

2 解题思路

  • 动态规划
    • 考虑将第i个元素当作当前数组最后一个元素进行划分，从第[i - k, i - 1]范围内枚举arr[j],用maVal记录以arr[j]结尾的最大数组值并求和从而进行更新d[i]
    • 状态转移方程：d[i] = max(d[i], d[j] + maxVal * (i - j))

3 复杂度分析

  • 时间复杂度
    • O(nk）
    • 数组有n个元素，每个元素倒序遍历前面k个元素
  • 空间复杂度
    • O(n)
    • dp数组空间大小n，记录对应位置的最大数组求和值

4 代码

// DP
class Solution:
    def maxSumAfterPartitioning(self, arr: List[int], k: int) -> int:
        n = len(arr)
        d = [0] * (n + 1)
        for i in range(1, n + 1):
            maxVal = arr[i - 1]
            for j in range(i - 1, max(-1, i - k - 1), -1):
                d[i] = max(d[i], d[j] + maxVal * (i - j))
                if j > 0:
                    maxVal = max(maxVal, arr[j - 1])
        return d[n]

1026-节点与其祖先之间的最大差值

2023-04-18
  • DFS
  • leetcode

1 题目描述

给定二叉树的根节点 root，找出存在于 不同 节点 A 和 B 之间的最大值 V，其中 V = |A.val - B.val|，且 A 是 B 的祖先。

（如果 A 的任何子节点之一为 B，或者 A 的任何子节点是 B 的祖先，那么我们认为 A 是 B 的祖先）

示例

输入：root = [8,3,10,1,6,null,14,null,null,4,7,13]
输出：7
解释： 
我们有大量的节点与其祖先的差值，其中一些如下：
|8 - 3| = 5
|3 - 7| = 4
|8 - 1| = 7
|10 - 13| = 3
在所有可能的差值中，最大值 7 由 |8 - 1| = 7 得出。

链接

题目来源：力扣（LeetCode）1026.节点与其祖先之间的最大差值

2 解题思路

  • DFS
    • 直接用DFS搜索二叉树的路径的所有节点，同时记录当前路径下成立父子关系节点的最大值ma和最小值mi
    • 统计每个节点对当前路径下mi与ma的绝对差值大小取最大值，最后DFS返回该最大值即可

3 复杂度分析

  • 时间复杂度
    • O(n）
    • 二叉树有n个节点需要遍历
  • 空间复杂度
    • O(n)
    • 最坏情况下，二叉树退化为链表，递归栈的空间为 O(n)

4 代码

// DFS
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxAncestorDiff(self, root: Optional[TreeNode]) -> int:
        def dfs(root, mi, ma):
            if root == None:
                return 0
            diff = max(abs(root.val - mi), abs(root.val - ma))
            mi = min(mi, root.val)
            ma = max(ma, root.val)
            diff = max(diff, dfs(root.left, mi, ma))
            diff = max(diff, dfs(root.right, mi, ma))
            return diff
        return dfs(root, root.val, root.val)

2409-统计共同度过的日子数

2023-04-17
  • leetcode
  • 前缀和
  • 简单模拟

1 题目描述

Alice 和 Bob 计划分别去罗马开会。

给你四个字符串 arriveAlice ，leaveAlice ，arriveBob 和 leaveBob 。Alice 会在日期 arriveAlice 到 leaveAlice 之间在城市里（日期为闭区间），而 Bob 在日期 arriveBob 到 leaveBob 之间在城市里（日期为闭区间）。每个字符串都包含 5 个字符，格式为 “MM-DD” ，对应着一个日期的月和日。

请你返回 Alice和 Bob 同时在罗马的天数。

你可以假设所有日期都在 同一个 自然年，而且 不是 闰年。每个月份的天数分别为：[31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31] 。

示例

输入：arriveAlice = "08-15", leaveAlice = "08-18", arriveBob = "08-16", leaveBob = "08-19"
输出：3
解释：Alice 从 8 月 15 号到 8 月 18 号在罗马。Bob 从 8 月 16 号到 8 月 19 号在罗马，他们同时在罗马的日期为 8 月 16、17 和 18 号。所以答案为 3 。

链接

题目来源：力扣（LeetCode）2409.统计共同度过的日子数

2 解题思路

  • 前缀和
    • 利用前缀和统计出每个月对应该年的天数，再加上天数得到日期具体是该年的哪一天
    • 用二个人离开日子的最小值减去到达日子的最大值即得到共同呆在罗马的日子

3 复杂度分析

  • 时间复杂度
    • O(1）
    • 前缀和处理12个月对应天数，时间复杂度为常数
  • 空间复杂度
    • O(1)
    • 一年只有12个月，存储这些天数空间复杂度为常数

4 代码

// 前缀和模拟
class Solution:
    def countDaysTogether(self, arriveAlice: str, leaveAlice: str, arriveBob: str, leaveBob: str) -> int:
        datesOfMonths = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
        prefixSum = [0]
        for date in datesOfMonths:
            prefixSum.append(prefixSum[-1] + date)

        arriveAliceDay = self.calculateDayOfYear(arriveAlice, prefixSum)
        leaveAliceDay = self.calculateDayOfYear(leaveAlice, prefixSum)
        arriveBobDay = self.calculateDayOfYear(arriveBob, prefixSum)
        leaveBobDay = self.calculateDayOfYear(leaveBob, prefixSum)

        return max(0, min(leaveAliceDay, leaveBobDay) - max(arriveAliceDay, arriveBobDay) + 1)


    def calculateDayOfYear(self, day: str, prefixSum: List[int]) -> int:
        month = int(day[:2])
        day = int(day[3:])
        return prefixSum[month - 1] + day

1157-子数组中占绝大多数的元素

2023-04-16
  • leetcode
  • 二分查找
  • 线段树
  • 随机化

1 题目描述

设计一个数据结构，有效地找到给定子数组的 多数元素 。

子数组的 多数元素 是在子数组中出现 threshold 次数或次数以上的元素。

实现 MajorityChecker 类:

MajorityChecker(int[] arr) 会用给定的数组 arr 对 MajorityChecker 初始化。
int query(int left, int right, int threshold) 返回子数组中的元素  arr[left…right] 至少出现 threshold 次数，如果不存在这样的元素则返回 -1。

链接

题目来源，力扣（LeetCode）1157.子数组中占绝大多数的元素

2 解题思路

  • 随机化
    • 题目本质是求区间[left, right]之间的众数
    • 众数出现次数>=区间长度，所以如果随机抽样该区间内的数，每次抽中众数的概率>= 1/2
    • 假设随机抽取30次，则一次都没抽中众数的概率<= (1/2)^k
    • 如果进行p次查询，每次查询相互独立，每次抽取k次则这p次查询结果都正确的概率为$$(1 - (\frac{1}{2})^k)^p.$$，当p = 10^4， k = 30时计算该表达式的结果近似为99.99%
    • 故认为该随机抽样得到区间[left, right]的正确性可以保证
  • 二分查找
    • 根据随机化原理得到区间的众数值x后，需要统计其出现的具体次数
    • 考虑使用哈希字典存储arr，统计每个元素出现的位置序列list
    • 在list中使用二分查找边界right和left出现的下标，相减后得到其出现次数cnt
    • 当cnt >= threshold时满足条件，如果cnt >= 区间长度的一半但却小于threshold则返回-1

3 复杂度分析

  • 时间复杂度
    • O(n + kqlog n）
    • 哈希表处理时间复杂度O(n),共进行q轮查询，每轮k次，每次二分的时间复杂度为O(log n)
  • 空间复杂度
    • 哈希存储空间复杂度O(n)

4 代码

// 随机化 + 二分查找
class MajorityChecker:
    k = 30

    def __init__(self, arr: List[int]):
        self.arr = arr
        self.cnt = defaultdict(list)
        for i, a in enumerate(arr):
            self.cnt[a].append(i)
            

    def query(self, left: int, right: int, threshold: int) -> int:
        arr_ = self.arr
        cnt_ = self.cnt

        length = right - left + 1
        for i in range(MajorityChecker.k):
            x = arr_[randint(left, right)]
            pos = cnt_[x]
            occ = bisect_right(pos, right) - bisect_left(pos, left)
            if occ >= threshold:
                return x
            elif occ * 2 >= length:
                return -1
        return -1

// 官方解法2：摩尔投票 + 线段树
class Node:
    def __init__(self, x: int = 0, cnt: int = 0):
        self.x = x
        self.cnt = cnt
    
    def __iadd__(self, that: "Node") -> "Node":
        if self.x == that.x:
            self.cnt += that.cnt
        elif self.cnt >= that.cnt:
            self.cnt -= that.cnt
        else:
            self.x = that.x
            self.cnt = that.cnt - self.cnt
        return self

class MajorityChecker:
    def __init__(self, arr: List[int]):
        self.n = len(arr)
        self.arr = arr
        self.loc = defaultdict(list)

        for i, val in enumerate(arr):
            self.loc[val].append(i)
        
        self.tree = [Node() for _ in range(self.n * 4)]
        self.seg_build(0, 0, self.n - 1)

    def query(self, left: int, right: int, threshold: int) -> int:
        loc_ = self.loc

        ans = Node()
        self.seg_query(0, 0, self.n - 1, left, right, ans)
        pos = loc_[ans.x]
        occ = bisect_right(pos, right) - bisect_left(pos, left)
        return ans.x if occ >= threshold else -1
    
    def seg_build(self, idx: int, l: int, r: int):
        arr_ = self.arr
        tree_ = self.tree

        if l == r:
            tree_[idx] = Node(arr_[l], 1)
            return
        
        mid = (l + r) // 2
        self.seg_build(idx * 2 + 1, l, mid)
        self.seg_build(idx * 2 + 2, mid + 1, r)
        tree_[idx] += tree_[idx * 2 + 1]
        tree_[idx] += tree_[idx * 2 + 2]

    def seg_query(self, idx: int, l: int, r: int, ql: int, qr: int, ans: Node):
        tree_ = self.tree

        if l > qr or r < ql:
            return
        
        if ql <= l and r <= qr:
            ans += tree_[idx]
            return

        mid = (l + r) // 2
        self.seg_query(idx * 2 + 1, l, mid, ql, qr, ans)
        self.seg_query(idx * 2 + 2, mid + 1, r, ql, qr, ans)

1024.不邻接植花

2023-04-15
  • leetcode
  • 标记遍历

1 题目描述

有 n 个花园，按从 1 到 n 标记。另有数组 paths ，其中 paths[i] = [xi, yi] 描述了花园 xi 到花园 yi 的双向路径。在每个花园中，你打算种下四种花之一。

另外，所有花园 最多 有 3 条路径可以进入或离开.

你需要为每个花园选择一种花，使得通过路径相连的任何两个花园中的花的种类互不相同。

以数组形式返回 任一 可行的方案作为答案 answer，其中 answer[i] 为在第 (i+1) 个花园中种植的花的种类。花的种类用  1、2、3、4 表示。保证存在答案。

链接

题目来源，力扣（LeetCode）1024.不邻接植花

2 解题思路

  • 标记+遍历
    • 先将所有花园的植花颜色设置为0， 然后使用list或defaultdict对花园连接情况进行记录
    • 遍历花园连接表，根据当前连接花园着色情况确定当前花园的颜色
    • 如果使用defaultdict存储花园连接则需要判断是否有孤立花园（没有与任何花园连接），有则将颜色0改为1即可

3 代码

// 个人解法，使用defaultdict存储
class Solution:
    def gardenNoAdj(self, n: int, paths: List[List[int]]) -> List[int]:
        flow_colors = [0] * n
        gardens = defaultdict(list)

        for i, j in paths:
            gardens[i].append(j)
            gardens[j].append(i)
        for g, l in gardens.items():
            colors = [False] * 5
            for i in l:
                colors[flow_colors[i - 1] - 1] = True
            for i, flag in enumerate(colors):
                if not flag:
                    flow_colors[g - 1] = i + 1
                    break
        for garden, color in enumerate(flow_colors):
            if color == 0:
                flow_colors[garden] = 1
        return flow_colors  

// 官方解法，使用二维list存储
class Solution:
    def gardenNoAdj(self, n: int, paths: List[List[int]]) -> List[int]:
        adj = [[] for i in range(n)]
        for path in paths:
            adj[path[0] - 1].append(path[1] - 1)
            adj[path[1] - 1].append(path[0] - 1)
        ans = [0] * n
        for i in range(n):
            colored = [False] * 5
            for vertex in adj[i]:
                colored[ans[vertex]] = True
            for j in range(1, 5):
                if not colored[j]:
                    ans[i] = j
                    break
        return ans
