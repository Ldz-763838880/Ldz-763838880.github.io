931-下降路径最小和

2023-07-13
  • leetcode
  • 动态规划
本文总阅读量次

1 题目描述

给你一个 n x n 的 方形 整数数组 matrix ，请你找出并返回通过 matrix 的下降路径 的 最小和 。

下降路径 可以从第一行中的任何元素开始，并从每一行中选择一个元素。在下一行选择的元素和当前行所选元素最多相隔一列（即位于正下方或者沿对角线向左或者向右的第一个元素）。
具体来说，位置 (row, col) 的下一个元素应当是 (row + 1, col - 1)、(row + 1, col) 或者 (row + 1, col + 1) 。

示例

输入：matrix = [[2,1,3],[6,5,4],[7,8,9]]
输出：13
解释：如图所示，为和最小的两条下降路径

链接

题目来源：力扣（LeetCode）931.下降路径最小和

2 解题思路

  • 简单DP
    • dp[i][j]表示到达当前matrix[i][j]时的最小路径和状态，
    • 初始状态，dp[0] = matrix[0]
    • 状态转移方程：
      • dp[i][j] = min(dp[i][j], min(dp[i-1][j], dp[i - 1][j - 1], dp[i - 1][j + 1]) + matrix[i][j]))
      • 这里需要考虑关于j是否越界的问题，分情况讨论
    • 最后返回结果dp[-1]里的最小值即为最小路径和

3 复杂度分析

  • 时间复杂度
    • DP
      • O(n^2)
      • 遍历一遍矩阵
  • 空间复杂度
    • DP
      • O(n^2)
      • 动态规划数组空间大小 n X n

4 代码

// DP 
class Solution:
    def minFallingPathSum(self, matrix: List[List[int]]) -> int:
        n = len(matrix)
        dp = [[inf] * n for _ in range(n)]
        for i in range(n):
            if i == 0 :
                for k in range(n):
                    dp[0][k] = matrix[0][k]
            else:
                for j in range(n):
                    if 0 < j < n - 1:
                        dp[i][j] = min(dp[i][j], min(dp[i - 1][j - 1], dp[i - 1][j], dp[i - 1][j + 1]) + matrix[i][j])
                    elif j == 0:
                        dp[i][j] = min(dp[i][j], min(dp[i - 1][j], dp[i - 1][j + 1]) + matrix[i][j])
                    else:
                        dp[i][j] = min(dp[i][j], min(dp[i - 1][j], dp[i - 1][j - 1]) + matrix[i][j])
        return min(dp[n - 1])

        # # # # # # # #
        dp = [matrix[0]]
        n = len(matrix[0])
        for i in range(1, n):
            cur = [0] * n
            for j in range(n):
                mn = dp[-1][j]
                if j > 0:
                    mn = min(mn, dp[i - 1][j - 1])
                if j < n - 1:
                    mn = min(mn, dp[i - 1][j + 1])
                cur[j] = mn + matrix[i][j]
            dp.append(cur)
        return min(dp[-1])
