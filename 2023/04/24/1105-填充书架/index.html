1105-填充书架

2023-04-24
  • leetcode
  • 动态规划
本文总阅读量次

1 题目描述

给定一个数组 books ，其中 books[i] = [thicknessi, heighti] 表示第 i 本书的厚度和高度。你也会得到一个整数 shelfWidth 。

按顺序 将这些书摆放到总宽度为 shelfWidth 的书架上。

先选几本书放在书架上（它们的厚度之和小于等于书架的宽度 shelfWidth ），然后再建一层书架。重复这个过程，直到把所有的书都放在书架上。

需要注意的是，在上述过程的每个步骤中，摆放书的顺序与你整理好的顺序相同。

例如，如果这里有 5 本书，那么可能的一种摆放情况是：第一和第二本书放在第一层书架上，第三本书放在第二层书架上，第四和第五本书放在最后一层书架上。
每一层所摆放的书的最大高度就是这一层书架的层高，书架整体的高度为各层高之和。

以这种方式布置书架，返回书架整体可能的最小高度。

示例

输入：books = [[1,1],[2,3],[2,3],[1,1],[1,1],[1,1],[1,2]], shelfWidth = 4
输出：6
解释：
3 层书架的高度和为 1 + 3 + 2 = 6 。
第 2 本书不必放在第一层书架上。

2 解题思路

  • 动态规划
    • 首先考虑使用dp存储放到第i本书时的最低高度
    • 其次考虑按照书的顺序进行遍历，当放到第i本书时，前j本书的最低高度dp[j] + j <= k <= i 区间书的最大高度就是当前第i本书的最低高度值
    • 动态规划递归方程为：dp[i] = min(dp[i], dp[j] + maxHig(j<= k <= i)]
    • 最后返回dp[n]就是n本书的最低高度值

3 复杂度分析

  • 时间复杂度
    • O(n^2）
    • n本书，每本书需要遍历前面的书
  • 空间复杂度
    • O(n)
    • 存储n本书的最低高度数组dp

4 代码

// DP
class Solution:
    def minHeightShelves(self, books: List[List[int]], shelfWidth: int) -> int:
        n = len(books)
        dp = [inf] *(n + 1)
        dp[0] = 0
        for i, b in enumerate(books):
            curWid, maxHig = 0, 0
            j = i
            while j >= 0:
                curWid += books[j][0]
                if curWid > shelfWidth:
                    break
                maxHig = max(maxHig, books[j][1])
                dp[i + 1] = min(dp[i + 1], dp[j] + maxHig)
                j -= 1
        return dp[n]
